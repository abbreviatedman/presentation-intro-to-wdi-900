# CI WDI 900-Hour Course Term 1 Intro for 09/2020


### To Serve Live Locally

```bash
npm install
npm run start
```


### To Build for Deployment

```bash
npm install
npm run build
```


### To Deploy As Webpage Using Surge

Install `surge` if necessary:
```bash
npm install -g surge
```
After building for deployment:
```bash
cd dist
surge
```
